# IT Software Request Policy

## Overview

This policy describes how employees request new software, the approval process, and the timeline for provisioning. All software installed on company devices must be approved through this process.

## Pre-Approved Software

The following software is pre-approved and can be provisioned within 1 business day without manager approval:

- Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams)
- Zoom
- Slack
- Google Chrome
- Adobe Acrobat Reader
- 7-Zip / WinRAR
- Notepad++
- VPN Client (Cisco AnyConnect)

To request pre-approved software, raise a ticket in the **IT Portal** at **it-portal.internal.com** under **Software → Standard Request**.

## Non-Standard Software Requests

Software not on the pre-approved list requires additional review. The process is:

1. **Submit request**: Log in to the IT Portal → **Software → Non-Standard Request**
2. **Provide justification**: Describe the business need and intended use
3. **Manager approval**: Your direct manager receives an approval request via email
4. **Security review**: The IT Security team reviews the software for vulnerabilities and licence compliance (up to 5 business days)
5. **IT provisioning**: Once approved, IT provisions the software within 2 business days
6. **Notification**: You receive an email confirmation when the software is ready

## Approval Timeline

| Software Type | Timeline |
|---|---|
| Pre-approved standard software | 1 business day |
| Non-standard (manager-approved only) | 3–5 business days |
| Non-standard requiring security review | 5–10 business days |

## Licencing

Employees must not install software using personal licence keys on company devices. All software must be licenced under the company's agreements. Using unlicensed software is a breach of the Code of Conduct and may result in disciplinary action.

## Software Removal

IT reserves the right to remove unapproved or unlicensed software from company devices without prior notice.

## Contact

Raise all software requests via the **IT Portal** at **it-portal.internal.com**. For urgent issues, contact the IT Helpdesk at **it-helpdesk@company.internal** or call extension **4000**.
